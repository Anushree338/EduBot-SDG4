# EduBot 🎓 - AI Study Assistant
### SDG 4: Quality Education

## 📌 Problem Statement
Millions of students lack access to personalized 
learning support. EduBot solves this by providing 
24/7 AI-powered study assistance for free.

## 🎯 Project Objective
Build an AI chatbot that helps students learn topics,
get quizzed, receive study tips and explore career paths.

## 🤖 How It Works
EduBot is built using Botpress with an AI Autonomous 
Node powered by LLM (Generative AI).

Students can:
- 📚 Learn any topic with simple explanations
- 🧠 Get quizzed on any subject
- 💡 Receive personalized study tips
- 🚀 Explore career guidance

## 🛠️ Technology Stack
- Platform: Botpress
- AI: Autonomous Node (LLM/Generative AI)
- Interface: Web Chat
